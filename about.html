Project description:
Attached is an internet project for the payment of a house committee in an apartment building.
Each cell in the table represents a tenant with details about the tenant obtained from JSON
When the tenant has a debt to the House Committee, the cell in the table is colored in red with a "pay" button and the amount payable.
By clicking the "pay" button, the tenant is transferred to a payment page with the payment amount and other parameters.
Clearing is done at a test terminal that simulates a credit transaction by sending a fictitious invoice - you can try.
After successful payment the tenant receives an electoral post invoice for the transaction.
Back to home page - The cell is painted green and updated the general debt of the building.
Page Admin - Allows adding and removing a tenant (updating the JSON).


The project includes the use of technologies:
Html, Css, Javascript, Jquery, Bootstrap
Ajax, Read form json, Rest API, Php
