תאור פרוייקט :
מצורף פרוייקט אינטרנטי לתשלום ועד-בית בבנין דירות.
כל תא בטבלה מייצג דייר עם הפרטים אודות הדייר שמתקבל מ JSON.
כאשר לדייר יש חוב ל ועד-בית התא בטבלה נצבע בצבע אדום עם כפתור "לשלם" והסכום לתשלום.
בלחיצה על כפתור "לשלם" הדייר מועבר לדף תשלום עם הסכום לתשלום
לאחר הרכישה הלקוח מקבל לדואר אלקטורני חשבונית על העסקה

הפרוייקט כולל שימוש בטכנולוגיות :
 Html , Css , Javascript , Jquery , Bootstrap
Ajax , Read form json , Rest API
